### 광둥어

- 음절 - 광둥어 발음 및 성조 데이터
- 한자 - 광둥어에서 자주 사용하는 한자 우선 작성 후, 한자 사전으로 데이터 늘려보자
- 단어 - 광둥어에서 자주 사용하는 단어. 중국어 단어는 우선 보류.
- 회화 - 간단한 회한 내용 작성. 일기 형식?
